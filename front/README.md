# 🛍️ E-Commerce Store - متجر إلكتروني متكامل


## 📋 نظرة عامة

متجر إلكتروني عصري متكامل تم بناؤه باستخدام React.js و Tailwind CSS، يتميز بتصميم جذاب وتجربة مستخدم سلسة. المتجر يدعم اللغة العربية بالكامل ويوفر جميع الميزات الأساسية للتسوق الإلكتروني.

## ✨ الميزات الرئيسية

### 🎨 **تصميم عصري وجذاب**
- تدرجات لونية احترافية (Indigo → Purple → Pink)
- تصميم متجاوب بالكامل (Responsive) يعمل على جميع الأجهزة
- تأثيرات حركية سلسة (Animations) باستخدام Tailwind
- واجهة مستخدم بديهية وسهلة الاستخدام

### 🛒 **نظام تسوق متكامل**
- **صفحة رئيسية** مع بانر دعائي وعداد تنازلي للعروض
- **عرض المنتجات** مع إمكانية التصفية حسب الفئات
- **نظام سلة تسوق** متطور مع إدارة الكميات
- **صفحة تفاصيل المنتج** مع خيارات (الألوان، المقاسات، الكمية)
- **إتمام الطلب** مع نموذج إدخال بيانات الشحن

### 🔐 **نظام مصادقة**
- تسجيل دخول المستخدمين
- إنشاء حساب جديد
- حماية المسارات الخاصة
- إدارة جلسات المستخدم

### 🚀 **ميزات تقنية**
- **React.js** مع Hooks (useState, useContext, useEffect)
- **React Router** للتنقل بين الصفحات
- **Context API** لإدارة حالة التطبيق (السلة، المصادقة)
- **Tailwind CSS** للتصميم والتنسيق
- **Lucide Icons** للأيقونات الحديثة

### 📱 **تجربة مستخدم محسنة**
- دعم كامل للغة العربية (RTL)
- شريط تنقل علوي ثابت
- قائمة جانبية للموبايل
- مؤشرات تحميل وتنبيهات
- عداد للسلة مع إشعارات

## 📦 المكونات الرئيسية

### 1. **Header**
- شريط تنقل علوي مع شعار المتجر
- قائمة جانبية للأجهزة المحمولة
- عداد للسلة وإدارة المستخدم

### 2. **Hero Section**
- بانر دعائي مع عرض خاص
- زرين رئيسيين للتسوق وتصفح الفئات
- علامة خصم متحركة

### 3. **Features**
- عرض المميزات (شحن مجاني، ضمان، دعم 24/7)
- أيقونات متحركة مع تدرجات لونية

### 4. **Categories**
- تصفية المنتجات حسب الفئة
- عرض شبكي للمنتجات مع صور وتفاصيل
- زر إضافة إلى السلة

### 5. **Offer Section**
- عروض خاصة مع عداد تنازلي
- عرض منتجات محدودة الوقت

### 6. **Cart**
- إدارة منتجات السلة
- تعديل الكميات وحذف المنتجات
- حساب المجموع الكلي

### 7. **Order**
- نموذج إدخال بيانات الشحن
- مراجعة الطلب قبل التأكيد
- تأكيد الطلب وإرساله

### 8. **Authentication**
- صفحة تسجيل دخول
- صفحة إنشاء حساب جديد
- تسجيل خروج آمن

### 9. **Product Details**
- عرض تفاصيل المنتج
- اختيار الألوان والمقاسات
- تحديد الكمية وإضافة للسلة

### 10. **Footer**
- نموذج تواصل معنا
- معلومات الاتصال
- روابط التواصل الاجتماعي

## 🛠️ التقنيات المستخدمة

- **React.js** - مكتبة بناء واجهات المستخدم
- **Tailwind CSS** - إطار عمل CSS للتصميم
- **React Router DOM** - للتنقل بين الصفحات
- **React Scroll** - للتمرير السلس للأقسام
- **Lucide React** - أيقونات حديثة
- **Context API** - إدارة حالة التطبيق
